# GESTIONNAIRE-DE-FILM
Application web interactive développée en JavaScript vanilla permettant d'afficher et de gérer une liste de films provenant d'une API externe. Ce projet a été réalisé dans le cadre d'un devoir maison noté et démontre la maîtrise des concepts fondamentaux de JavaScript.
Application web interactive développée en JavaScript vanilla permettant d'afficher et de gérer une liste de films provenant d'une API externe. Ce projet a été réalisé dans le cadre d'un devoir maison noté et démontre la maîtrise des concepts fondamentaux de JavaScript.

🚀 Fonctionnalités
Principales
Chargement des données : Récupération automatique des films via l'API JSONPlaceholder

Affichage dynamique : Présentation des films sous forme de cartes élégantes avec ID, titre et contenu

Recherche en temps réel : Filtrage des films par titre (insensible à la casse)

Compteur en direct : Affichage du nombre de films actuellement visibles

Bonus
Filtrage par ID pair : Bouton dédié pour n'afficher que les films avec des identifiants pairs

🛠️ Technologies Utilisées
HTML5 : Structure sémantique de la page

CSS3 : Design moderne avec dégradés, animations et responsive design

JavaScript (Vanilla) :

Manipulation du DOM

Gestion des événements

Appels API avec Fetch

Manipulation de données JSON

Méthodes de tableaux (filter, forEach)

Programmation orientée objet
